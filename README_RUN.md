# 如何运行本项目

这是一个基于 Vite + React + TypeScript 的前端项目。

## 前置要求

你需要安装 [Node.js](https://nodejs.org/) (建议安装 LTS 版本)。

## 运行步骤

1.  **安装依赖**
    打开终端，运行以下命令：
    ```bash
    npm install
    ```

2.  **启动开发服务器**
    依赖安装完成后，运行：
    ```bash
    npm run dev
    ```
    启动后，终端会显示一个本地访问地址（通常是 `http://localhost:5173`），在浏览器中打开即可。

## 其他命令

- **构建项目**：`npm run build` (生成的静态文件位于 `dist` 目录)
- **预览构建结果**：`npm run preview`

## 常见问题

- 如果终端提示 `npm` 不是内部或外部命令，请确保已正确安装 Node.js 并将其添加到了系统环境变量中。
